🚀 QUICK START (5 STEPS)

1. Start the app:
   cd /home/quint/aco-bot
   npm start

2. Create a profile:
   - Go to Profiles page
   - Click "+ New Profile"
   - Fill in your info (name, email, address, payment)
   - Save

3. Add your proxies:
   - Click on the profile you created
   - Click "+ Import Proxies"
   - Select "ISP" for Walmart or "Residential" for Shopify
   - Paste your proxy credentials:
     user:pass@gate.smartproxy.com:20000
     user:pass@gate.smartproxy.com:20001
   - Click Import

4. Test proxies:
   - Click "Test" on each proxy
   - Green ✅ = Working
   - Red ❌ = Delete it

5. Create tasks:
   - Go to Dashboard
   - Create task
   - Select your profile (with proxies)
   - Run task!
